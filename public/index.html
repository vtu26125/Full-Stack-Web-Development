About

Electronics Inventory Management System

This system helps electronics stores track products, suppliers, sales, and purchases with real-time stock updates. Add products with category, product type, and configuration dropdowns, monitor low stock, and view reports.

Key Features

  • Inventory tracking with low stock alerts
  • Sales and purchase history
  • Supplier and category management
  • Configurable product types (e.g., Laptop, Router)

Built With

  • Node.js + Express
  • PostgreSQL
  • Vanilla JavaScript
  • Responsive dashboard UI
